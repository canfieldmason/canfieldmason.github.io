Shell Eco-Marathon Chassis Structural Analysis

Work in progress — student competition vehicle

This project documents the ongoing structural analysis and development of a prototype vehicle chassis for Shell Eco-Marathon. The focus is on establishing a robust analysis workflow to evaluate chassis stiffness, load paths, and structural efficiency while supporting lightweight vehicle design objectives.

Current Status:
  • Baseline chassis geometry under development
  • Preliminary load cases being defined
  • Analysis workflow planning and validation in progress

Project Objectives

  • Develop a structurally efficient chassis suitable for Shell Eco-Marathon competition
  • Apply finite element analysis to evaluate stiffness and load distribution
  • Compare alternative chassis concepts based on structural performance and mass
  • Establish a repeatable analysis workflow for future design iterations

Planned Analysis Scope

The chassis analysis will include a sequence of representative load cases commonly used in lightweight vehicle design:

  • Torsional stiffness evaluation
  • Cornering and braking load cases
  • Driver and component mass loading
  • Localized stress evaluation at joints and mounting interfaces

Geometry simplification, mesh refinement strategies, and boundary condition assumptions will be documented to clearly communicate modeling decisions.

Tools & Methods

  • CAD: Fusion 360
  • Simulation: ANSYS Mechanical (Static Structural)
  • Modeling approach: Beam / shell / solid elements as appropriate

Next Steps

  • Finalize baseline chassis geometry
  • Define initial loading and constraint assumptions
  • Generate first-pass structural results for stiffness benchmarking
  • Iterate design based on analysis findings

This page will be updated as analysis results and design iterations progress.
