Shell Eco-Marathon Chassis Structural Analysis

Ongoing design and analysis project — student competition vehicle

Project Overview

This project documents the structural development of a prototype vehicle chassis for Shell Eco-Marathon. The focus is on applying finite element analysis to guide a lightweight, efficient, and manufacturable design suitable for a student-built competition vehicle.

Unlike the guided Formula SAE and Baja SAE analyses, this project represents a fully original design effort beginning with concept generation and progressing through iterative modeling, simulation, and refinement.

Tools & Methods

CAD: SOLIDWORKS
Simulation: ANSYS Mechanical
Approach: Beam, shell, and solid element modeling as appropriate
Techniques: Load case development, stiffness benchmarking, design iteration

Current Status
  • Baseline chassis geometry under development
  • Initial load cases being defined
  • Preliminary analysis workflow planning in progress
  • Material and manufacturing constraints being incorporated

Engineering Objectives

  • Design a structurally efficient chassis optimized for minimal mass
  • Use finite element analysis to evaluate stiffness and load distribution
  • Compare alternative structural concepts and joint configurations
  • Establish a repeatable analysis workflow for future vehicle iterations
  • Balance structural performance with manufacturability and serviceability

Planned Analysis Scope

The analysis plan is structured around representative operating scenarios for a lightweight efficiency vehicle:

  • Torsional stiffness evaluation for handling performance
  • Cornering and braking load cases
  • Driver and component mass loading
  • Localized stress evaluation at joints and mounting interfaces
  • Mounting analysis for suspension, steering, and drivetrain systems

Modeling assumptions, boundary conditions, and mesh strategies will be documented to maintain transparency and repeatability throughout the design process.

Planned Workflow

  1. Develop initial concept geometry in SOLIDWORKS
  2. Simplify geometry for efficient FEA representation
  3. Create baseline finite element model in ANSYS
  4. Apply standardized load cases
  5. Evaluate stiffness, stress, and mass efficiency
  6. Iterate geometry based on results
  7. Refine design for manufacturability and assembly

Next Steps

  • Finalize first-pass chassis layout
  • Implement initial boundary condition definitions
  • Generate baseline structural results
  • Establish quantitative stiffness and mass benchmarks
  • Begin iterative design optimization cycle
