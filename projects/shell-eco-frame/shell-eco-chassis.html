Reference — Model 5 (Previous Competition Year)

The Model 5 chassis from a prior competition cycle serves as a structural and packaging reference. Notably shorter and stubbier with a rounder, kayak-like cross-section, it reflects an earlier design philosophy. This year's frame departs significantly — longer wheelbase, more angular primary structure, and revised rollbar geometry in line with current SEM regulations.

Model 5 reference frame — shorter, stubbier, and rounder than the current design. Retained as a packaging and geometry benchmark.

Shell Integration — Senior Team CFD/FEA Reference Geometry

The senior team provided a temporary aluminum shell mated to the chassis for use in CFD and FEA groundwork. The shell geometry reflects the vehicle's aerodynamic outer envelope and is useful for early drag estimation and structural load input definition.

Following team review, the decision was made to transition FEA and CFD workflows to the SEM-provided Altair and Siemens suite rather than ANSYS — aligning with official competition toolchain support and long-term team capability building. The shell geometry remains a valid reference asset for both domains.

Chassis with senior team temporary aluminum shell — used for initial CFD and FEA geometry setup. Team has since pivoted to SEM-provided Altair + Siemens toolchain.

Planned Analysis Scope

With the FEA-ready geometry established, the following load cases are planned once the Altair suite is integrated:

  • Torsional stiffness evaluation — opposing vertical loads at suspension pickups
  • Bending stiffness — distributed driver and powertrain mass loading
  • Braking and cornering load cases — longitudinal and lateral acceleration inputs
  • Joint and mounting interface stress — rollbar base and bulkhead attachments
  • SEM structural compliance checks — per competition rules and standards
Toolchain Note Analysis will be conducted in the SEM-provided Altair suite (FEA) and Siemens suite (CFD), replacing ANSYS Mechanical as the primary solver. This aligns with official SEM competition support and enables direct correlation with other competing teams' workflows. All boundary conditions, mesh strategies, and assumptions will be documented for repeatability.

Engineering Objectives

  • Deliver a lightweight, structurally efficient frame optimized for the SEM competition envelope
  • Validate design using Altair FEA correlated against the MATLAB/Simulink road load model
  • Improve stiffness-to-mass ratio relative to the Model 5 reference chassis
  • Ensure full compatibility with body shell, drivetrain packaging, and suspension geometry
  • Establish documented analysis workflows for future JMU teams
